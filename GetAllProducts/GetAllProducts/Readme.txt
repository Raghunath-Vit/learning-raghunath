Product Details Component
This project demonstrates the Agile methodology by creating a ProductDetails component that displays detailed information about a product based on its ID. The component uses the Fakestore API to fetch and display product details.

Task Overview:
The objective is to develop a ProductDetails component that displays information for a product with a given ID, accessible via a URL like /productdetails/12. 

The details are fetched from the Fakestore API.



API Endpoint
To fetch product details, use the following endpoint:


https://fakestoreapi.com/products/{id}
Replace {id} with the desired product ID.


Example
For product ID 12, the endpoint would be:
https://fakestoreapi.com/products/12


Data Displayed
The ProductDetails component displays:
Product Title
Product Description
Product Price
Product Image
Product Rating
